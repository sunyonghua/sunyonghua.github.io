字符串不可变是指字符串的成员函数不会去改变原始值,而是创建并返回一个新的字符串,而数组的成员函数都是在其原始值上操作

eg:

let str = "yonghua"
str.toUpperCase()
str.slice(1,2)
console.log(str) // "yonghua"

可借用数组的不可变成员函数，不能借用数组的可变成员函数

eg:

let str = "yonghua"
Array.prototype.join.call(str, '-')
console.log(str) // 'y-o-n-g-h-u-a'
Array.prototype.push.call(str,'sun')
// 报错
